```
│  └─...
├─doc  # document
│  └─...
├─lib  # node libs
│  └─...
├─system  # system modules
│  └─...
├─autorunx.py  # start module
├─globals.py  # global module
├─setup.py  # setup
|
```

**Notice**: Make sure module names in 'lib' are unique by naming them according to the rules.
